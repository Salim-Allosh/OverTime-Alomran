## 📋 الخطوات التالية (بالترتيب)

### المرحلة 1: تحديث قاعدة البيانات
```bash
# تشغيل سكريبت SQL
mysql -u root -p AlomranReportsDB < backend-api/update_account_roles.sql

# أو استخدام Python script
cd backend-api
python add_sales_manager_column.py  # إذا لم يتم تشغيله بعد
```

### المرحلة 2: تحديث Frontend - أنواع الحسابات
1. تحديث `AdminPage.jsx`:
   - إضافة checkboxes لجميع أنواع الحسابات
   - عرض نوع الحساب في الجدول
   - منع إنشاء Backdoor accounts من الواجهة

2. تحديث `Layout.jsx`:
   - إظهار/إخفاء الصفحات حسب الصلاحيات
   - استخدام `userInfo.role` لتحديد الصفحات المتاحة

3. تحديث جميع الصفحات:
   - تطبيق `can_edit()` للصفحات التي تحتاج تعديل
   - تطبيق `assert_branch_access()` للصفحات المحددة بفرع

### المرحلة 3: إنشاء الصفحات الجديدة

#### 1. صفحة التقارير اليومية للمبيعات (`/daily-sales-reports`)
- إدخال تقارير مبيعات يومية
- مرتبطة بالفروع وموظفي المبيعات
- PDF export

#### 2. صفحة التقارير الشهرية للعقود (`/monthly-contract-reports`)
- عرض وتحليل العقود شهرياً
- حسب الفرع
- PDF export

#### 3. تحديث صفحة الإحصائيات (`/statistics`)
- إضافة رسوم بيانية (Chart.js أو Recharts)
- مؤشرات أداء
- تجميع من جميع التقارير

#### 4. صفحة صافي الأرباح (`/net-profit`)
- الدخل - المصاريف = صافي الربح
- فلاتر زمنية
- PDF export

#### 5. صفحة الميزانية (`/budget`)
- الدخل العام والشهري
- إضافة مصاريف
- فلاتر زمنية
- PDF export

#### 6. صفحة إدارة موظفي المبيعات (`/sales-staff`)
- إضافة/تعديل موظفي المبيعات
- مرتبطة بالفروع
- للـ Sales Manager فقط
