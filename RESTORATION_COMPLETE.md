## 🚀 خطوات التشغيل

### 1. تحديث قاعدة البيانات

#### الطريقة الأولى: استخدام Python Script
```bash
cd backend-api
python add_sales_manager_column.py
```

#### الطريقة الثانية: استخدام SQL
1. افتح phpMyAdmin
2. اختر قاعدة البيانات `AlomranReportsDB`
3. انسخ محتوى `backend-api/add_sales_manager_column.sql`
4. الصقه في تبويب SQL واضغط Execute

### 2. تشغيل المشروع
```bash
run.bat
```

### 3. إنشاء حسابات مديري المبيعات
1. سجل الدخول كـ Super Admin
2. اذهب إلى `/admin`
3. اضغط "إنشاء حساب جديد"
4. اختر الفرع
5. ضع علامة على "مدير مبيعات"
6. احفظ
