## 📋 الخطوات التالية (للمستخدم)

### 1. تحديث قاعدة البيانات
```bash
# في phpMyAdmin أو MySQL
mysql -u root -p AlomranReportsDB < backend-api/create_all_tables.sql
```

### 2. تثبيت المكتبات الجديدة
```bash
cd frontend
npm install
```

### 3. تشغيل النظام
```bash
# من المجلد الرئيسي
run.bat
```
