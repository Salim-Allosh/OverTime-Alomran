# 🎨 نظام التصميم المعتمد - Design System

## ⚠️ مرجع إلزامي لجميع التعديلات والإنشاءات الجديدة

---

## 1️⃣ الهوية البصرية المعتمدة (Design System Core)

### الخط (Typography)

**الخط الوحيد المستخدم:** Cairo  
**Fallback:** sans-serif

**خصائص عامة:**
- Anti-aliasing مفعل
- Line-height منخفض لزيادة كثافة البيانات

#### أحجام الخط

| الاستخدام | الحجم |
|----------|-------|
| عنوان الصفحة | 16px |
| عنوان الجدول/البطاقة | 13px |
| رأس الأعمدة | 12px |
| بيانات الجدول | 11px |
| أزرار | 12px |
| تنبيهات | 11–12px |

---

### لوحة الألوان الرسمية (Color Palette)

#### الألوان الأساسية (إلزامية)

| الغرض | اللون |
|-------|------|
| الخلفية العامة | `#F5F2F2` |
| النص الأساسي | `#2B2A2A` |
| اللون الرئيسي (Primary) | `#5A7ACD` |
| لون التمييز (Accent) | `#FEB05D` |

**❗ لا تستخدم ألوان أخرى - فقط هذه الألوان الأربعة**

---

## 2️⃣ أسلوب التصميم العام (Visual Style)

### Enterprise Classic UI - تصميم كلاسيكي أنيق

**المبادئ الأساسية:**
- ✅ جداول واضحة ذات حدود واضحة
- ✅ بطاقات (Cards) بسيطة بدون ظلال
- ✅ عناصر مضغوطة (غير متباعدة)
- ✅ مساحات بيضاء قليلة
- ✅ تصميم كلاسيكي أنيق
- ❌ بدون ألوان فاقعة
- ❌ بدون تدرجات
- ❌ بدون ظلال كبيرة

### الحواف (Borders)

- **Border-radius:** `3px` (حد أدنى)
- **Border color:** `#D1D1D1` أو `#2B2A2A` (حدود واضحة)
- **Border width:** `1px` أو `2px` (للتمييز)

---

## 3️⃣ تخطيط الواجهة (Page Layout)

### بنية الصفحة

```
┌─────────────────────────────────────┐
│ Top Bar (لون أزرق غامق)              │
├─────────────────────────────────────┤
│ Content Area (خلفية #F5F2F2)         │
│ ┌─────────────────────────────────┐ │
│ │ Card/Table Container             │ │
│ │ (خلفية بيضاء، حدود واضحة)        │ │
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

### خصائص عامة

- ✅ المسافات بين العناصر: `0.5rem - 0.75rem` (مضغوطة)
- ✅ Padding داخل البطاقات: `0.75rem - 1rem`
- ✅ المحتوى يبدأ مباشرة
- ✅ تصميم مضغوط لعرض أكبر كمية بيانات

---

## 4️⃣ توصيف الجداول (Table Specification)

### نوع الجداول

✅ **HTML Table تقليدي**

### شكل الجدول العام

- **العرض:** 100%
- **الخلفية:** أبيض
- **الحدود:** واضحة `1px solid #2B2A2A` أو `#D1D1D1`
- **يدعم:**
  - Scroll أفقي
  - Scroll عمودي
  - قابل للطباعة مباشرة

### رأس الجدول (Table Header)

- **الخلفية:** `#F5F2F2`
- **الخط:** Cairo SemiBold – `12px`
- **اللون:** `#2B2A2A`
- **Border-bottom:** `2px solid #2B2A2A`
- **ارتفاع الصف:** `32px`
- **Padding:** `6px 8px`

### صفوف البيانات (Rows)

- **ارتفاع الصف:** `28px` (مضغوط)
- **Padding داخل الخلية:** `6px 8px`
- **Border-bottom:** `1px solid #D1D1D1`
- **Hover:**
  - Background: `#F5F2F2`

### خلايا البيانات (Cells)

- **خط:** Cairo Regular – `11px`
- **اللون:** `#2B2A2A`
- **محاذاة:**
  - أرقام: Right
  - نصوص: Left
- ❌ لا يوجد Wrap افتراضي (nowrap)
- ✅ يظهر Ellipsis عند اللزوم

---

## 5️⃣ البطاقات (Cards)

### مواصفات البطاقة

- **الخلفية:** أبيض
- **الحدود:** `1px solid #2B2A2A` (واضحة)
- **Border-radius:** `3px`
- **Padding:** `0.75rem`
- **Margin-bottom:** `0.5rem` (مضغوط)
- **Box-shadow:** لا يوجد (أو `0 1px 2px rgba(0,0,0,0.05)` للعمق البسيط)

### استخدام البطاقات

- ✅ للإحصائيات (Statistics Cards)
- ✅ للفلاتر (Filter Cards)
- ✅ للنماذج (Form Cards)
- ✅ للمحتوى المجمع

---

## 6️⃣ الفلاتر وأدوات التحكم

- ✅ توضع في بطاقة صغيرة مضغوطة
- ✅ عناصر صغيرة ومضغوطة
- ✅ المسافات: `0.5rem`

### الأزرار

- **Flat Buttons**
- **Primary:** `#5A7ACD` (خلفية) + أبيض (نص)
- **Secondary:** أبيض (خلفية) + `#2B2A2A` (نص) + حدود
- **ارتفاع الزر:** `30px`
- **Padding:** `0.4rem 0.75rem`
- **Border:** `1px solid #2B2A2A` (للأزرار الثانوية)

### زر تحميل PDF

- ✅ موجود في كل صفحة تقرير
- ✅ أيقونة صغيرة + نص
- ✅ لا يلفت الانتباه بصريًا

---

- ✅ الاحتفاظ بـ:

## 📋 قواعد الاستخدام

1. **عند إنشاء صفحة جديدة:** استخدم هذا النظام بالكامل
2. **عند تعديل صفحة موجودة:** طابقها مع هذا النظام
3. **الجداول:** HTML Table فقط، حدود واضحة
4. **البطاقات:** بسيطة، حدود واضحة، بدون ظلال كبيرة
5. **الألوان:** استخدم الألوان الأربعة المحددة فقط
6. **المسافات:** مضغوطة (0.5rem - 0.75rem)
7. **التصميم:** كلاسيكي أنيق، بدون ألوان فاقعة
